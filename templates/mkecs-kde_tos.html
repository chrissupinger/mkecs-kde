mkecs-kde: Terms of Service
1. Why a ToS?
Transparency. Period. It's important to be honest about what this service is meant to do and what's to be expected during its operation.
2. What does it do?
This service presents an innovative application and data science centric method of mapping and predicting crime using kernel density estimation (KDE). It enables access to over five years of aggregated crime data and visualization through the lenses of a variety of parameterized factors. Besides individual use, this project is made available to study groups to examine opinion variances of patrol distributions.
3. What data is considered?
The dataset consists of millions of observations derived from the Wisconsin Incident-Based Reporting System (WIBRS) spanning 2005 to present day. Although the majority of that data range is currently unavailable, data points are regularly added to the dataset in order to make them available to this service.
4. How does it work?
As a user, you're presented with the ability to filter and visualize the dataset through two sets of parameters which also consist of supplemental classifications. A selection for each classification is required to execute a KDE on the dataset. Since KDEs are resource intensive, it can take an extended amount of time to generate a query. 5. What about privacy?
User privacy is of utmost importance. When a user confirms that they've read this ToS, a randomized session ID is generated and assigned to the user and their HTTP cookie. No personal information, usernames, passwords or user inputed data is needed or recorded. A session ID is solely used to generate a unique key to store a user's session. This data is reset upon a user's conscious action to exit the session, when the browser is closed or when a user's internet history is deleted.
6. Is there anything else?
Yes! When all else fails, you run into something you don't expect or you're just not sure about something, please feel free to contact me. The more information you can provide, about a service anomaly, the better.
